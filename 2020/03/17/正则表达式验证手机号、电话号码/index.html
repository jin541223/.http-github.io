正则表达式验证手机号、固话

发表于 2020-03-17

规则

固话

区号是以 0 开头的，后面是 2-3 位，正则表达式0\d{2,3}。固定电话号码由 7~8 位数字，则使用表达式\d{7,8}来进行匹配。固话的组合方式可能是“区号-号码”或是“区号号码”

bash /0\\d{2,3}[-]?\\d{7,8}|0\\d{2,3}\\s?\\d{7,8}/ bash

手机号

手机号是 11 位数，并且以数字 1 开头

bash /^[1](([3][0-9])|([4][5-9])|([5][0-3,5-9])|([6][5,6])|([7][0-8])|([8][0-9])|([9][1,8,9]))[0-9]{8}$/ bash

示例

``` bash
function verifyMobile($mobile)
{
$mobileRule = “/^1[0-9]{8}$/“;
$telRule = “/0\d{2,3}[-]?\d{7,8}|0\d{2,3}\s?\d{7,8}|13[0-9]\d{8}|15[1089]\d{8}/“;

    if (!preg_match($mobileRule, $mobile) && !preg_match($telRule, $mobile)) {
        return false;
    }

    return true;
}

$isVerifyPass = verifyMobile('1856562522'); 
var_dump($isVerifyPass);

``` bash
