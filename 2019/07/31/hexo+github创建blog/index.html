hexo + github + node + npm 创建个人博客

hexo 3.9.0，node 8.11.1，npm 5.7.0

安装hexo

命令行输入：npm install -g hexo-cli
![avatar][installHexo]

初始化hexo

命令行输入：hexo init blog
![avatar][initHexo]

安装依赖包

命令行输入：npm install
![avatar][installNpm]

站点配置

发布到git需要更改

查看默认站点

生成文件 hexo g 或 hexo generate

启动服务 hexo s 或 hexo server[浏览器输入localhost:4000]

部署到github

# hexo github node npm
