explain SQL语句字段说明

id

SELECT查询标识符，每个SELECT会自动分配一个唯一的标识符

select_type

SELECT查询的类型

table

表示查询涉及到的表或衍生表

partitions

匹配的分区

type

join链接访问类型，依次由好到差

possible_keys

本次查询中可能使用到的索引【有些索引在此字段中出现也不一定就被使用，实际使用索引要根据key的值决定】

key

本次查询中实际使用到的索引

key_len

查询优化器使用索引的字节数【用于评估组合索引是否被完全使用或只有最左部分字段被使用到】

ref

与key一起被使用的字段

rows

预估本次查询一共扫描了多少行【值越小越好，表明效率的好坏】

filtered

本次查询条件过滤的数据的百分比

Extra

额外信息
