### git下载主题包
``` bash
$ git clone https://github.com/iissnan/hexo-theme-next themes/next
```
下载完成themes目录下会增加next目录

### 启用主题
更改目录站点下\_config.yml
``` bash
theme: next
```

### 主题配置项
目录themes\/next\/\_config.yml，打开任一项即可
``` bash
$ # Schemes
$ scheme: Muse
$ scheme: Mist
$ scheme: Pisces
#scheme: Gemini
```
